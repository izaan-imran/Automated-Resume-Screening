JobJive AI: Automated Resume Screening SystemJobJive AI is an advanced recruitment tool that automates the process of screening resumes. Using a hybrid approach of Machine Learning and Rule-Based logic, it classifies resumes into 24 distinct professional categories, helping HR teams filter thousands of applications in seconds.  📖 Project OverviewThe primary goal of this project is to reduce the manual effort and unconscious bias involved in the initial hiring phase. By leveraging Natural Language Processing (NLP), the system reads the content of a resume and predicts the most suitable job role for the candidate.  ⚙️ How It Works (Pipeline)The system follows a structured data science pipeline to ensure accuracy:Text Extraction: Raw text is extracted from PDF or plain-text files using pdfplumber.  Preprocessing: The text undergoes deep cleaning, including lowercase conversion, removal of URLs/emails, and stripping of English stopwords using NLTK.  Vectorization: Cleaned text is converted into numerical data using TF-IDF Vectorization, focusing on the 3,000 most informative terms.  Hybrid Classification: The vector is passed through both Machine Learning models and a specific Rule-Based layer to determine the final category.  🤖 Why These Models?We implemented an Ensemble Majority Voting System involving four models to ensure that no single model's error dominates the final decision:  Random Forest (75.25%): Our best performer. It uses 200 decision trees to handle noisy text data and reduce overfitting.  Linear SVC (72.84%): Highly effective for high-dimensional text features where classes are separated by a clear margin.  Logistic Regression (69.82%): Used as a strong baseline for linear text classification.  Naive Bayes (55.94%): Included for its speed and unique probability-based perspective on sparse TF-IDF vectors.  🛠️ Rule-Based Override LayerA common challenge in ML-based screening is the overlap between Data Science and Digital Media terminology (e.g., both use "content" or "python"). To solve this:  The system scans for 30 specific IT signals (e.g., TensorFlow, PyTorch, Pandas, SQL).  If 3 or more strong signals are found and they outweigh "noise" words, the system automatically overrides the ML models to classify the resume as INFORMATION-TECHNOLOGY with 100% confidence.  🚀 Deployment & LinksThe project is deployed as an interactive Streamlit web application.
